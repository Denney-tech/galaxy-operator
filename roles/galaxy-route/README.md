Galaxy Route
===========

A role to setup Galaxy routes in Openshift, yielding the following objects:

* Ingress/Route

Requirements
------------

Requires the `openshift` Python library to interact with Openshift: `pip install openshift`.

Dependencies
------------

collections:

  - kubernetes.core
  - operator_sdk.util
